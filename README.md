# KernelSense
> **"Your operating system, understood and foreseen."**

KernelSense is an AI-augmented, real-time operating system observatory. It collects live kernel and process telemetry through a tiered, consent-gated instrumentation pipeline, and surfaces predictions (like resource saturation and memory leaks) via time-series forecasting and anomaly-detection models.

## Project Structure
- `backend/` - FastAPI backend, AI engine, and data collection
- `frontend/` - Next.js frontend dashboard
- `docs/` - Architecture Decision Records (ADRs) and design documentation
